# Les Caisses du Léman - Landing Page

Une landing page moderne et responsive créée avec React et Vite, inspirée des Alpes suisses et de la région du Léman.

## 🎨 Design

Le design reflète l'identité visuelle des Caisses du Léman avec :
- **Couleurs** : Bleu suisse (#0055A4), Rouge (#DC143C) et Blanc
- **Thème** : Inspiré des Alpes et de la région du Léman
- **Logo** : Intégration du logo officiel avec ses formes géométriques caractéristiques
- **Éléments visuels** : Montagnes, lac Léman, croix suisse

## 🚀 Installation

1. Installez les dépendances :
```bash
npm install
```

## 💻 Développement

Lancez le serveur de développement :
```bash
npm run dev
```

La page sera accessible sur `http://localhost:5173`

## 🏗️ Build

Pour créer une version de production :
```bash
npm run build
```

Les fichiers seront générés dans le dossier `dist/`

## 📦 Prévisualisation

Pour prévisualiser la version de production :
```bash
npm run preview
```

## 🎨 Fonctionnalités

- Design moderne et responsive avec thème alpin/suisse
- Logo intégré dans le header
- Navigation fluide
- Section hero avec illustration du lac Léman et des Alpes
- Présentation des services avec icônes thématiques
- Section à propos avec statistiques animées
- Call-to-action avec dégradé aux couleurs du logo
- Footer complet avec informations de contact

## 🛠️ Technologies

- React 18
- Vite
- CSS3 (sans framework)
- SVG pour les logos et illustrations
